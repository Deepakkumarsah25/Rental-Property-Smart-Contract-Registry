# Rental Property Smart Contract Registry

## Project Description
A decentralized blockchain-based registry for rental properties that enables secure, transparent, and automated rental agreements. This smart contract platform eliminates intermediaries in the rental process, providing direct interaction between property owners and tenants with built-in payment escrow and security deposit management.

## Project Vision
To revolutionize the rental property industry by creating a trustless, transparent, and efficient ecosystem where property owners and tenants can interact directly without intermediaries. Our vision is to reduce rental fraud, minimize disputes, and create a global, accessible rental marketplace powered by blockchain technology.

## Key Features

### 1. Property Registration & Management
- **Property Listing**: Owners can register properties with detailed information including location, description, and pricing
- **Availability Control**: Dynamic availability status updates
- **Secure Ownership**: Blockchain-verified property ownership

### 2. Smart Rental Agreements
- **Automated Agreements**: Self-executing rental contracts with predefined terms
- **Date Management**: Flexible start and end date configurations
- **Payment Calculation**: Automatic calculation of rental amounts based on duration

### 3. Secure Payment Handling
- **Escrow System**: Secure holding of rental payments and security deposits
- **Automated Transfers**: Timely payment distribution to property owners
- **Deposit Protection**: Secure handling and return of security deposits
- **Transparent Transactions**: All payments recorded on blockchain

### 4. User Management
- **Property Owner Dashboard**: Track owned properties and rental history
- **Tenant Portal**: Manage current and past rentals
- **Role-based Access**: Secure permission system for all operations

- **Smart Contract Language**: Solidity ^0.8.19
- **Key Standards**: ERC-20 for payments, potential ERC-721 for property NFTs
- **Security**: Comprehensive modifier checks, reentrancy protection, and emergency functions

## Security Features

- Comprehensive input validation

## Getting Started
1. Deploy the smart contract on your preferred Ethereum-compatible network
2. Property owners can register properties using `registerProperty()`
3. Tenants can create rental agreements with `createRentalAgreement()`
4. Manage property availability with `updatePropertyAvailability()`

   Contract Address: 0xD94f465cc5F82d7A5Ea886e048CaF8bf1f76B884
6. Process security deposit returns with `returnSecurityDeposit()`
